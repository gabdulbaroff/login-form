Demo-app for login form with React and TypeScript.

## Stack

- React 19
- TypeScript
- React Router
- Vite
- CSS Modules
- Lottie (animations)
- PostCSS Custom Media

## Functionality

- Login form with validation
- Password visibility toggle
- Authenticated routes (require authorization)
- Adaptive design

## Project structure

```
src/
├── api/          # API
├── components/   # UI components
├── context/      # React Context (authorization)
├── icons/        # SVG icons
├── pages/        # Pages
├── router/       # Router
└── styles/       # CSS variables
```

## Scripts

```bash
npm run dev      # Run dev server
npm run build    # Build project
npm run lint     # Check linter
npm run deploy   # Deploy on GitHub Pages
```
